# Web Server Project

This project contains Bash scripts for server management.

## 0-transfer_file

This script transfers a file from a client machine to a remote server using SCP.

### Usage:
./0-transfer_file PATH_TO_FILE IP USERNAME PATH_TO_SSH_KEY

## 1-install_nginx_web_server
Installs nginx and configures it to serve a page containing "Holberton School".
